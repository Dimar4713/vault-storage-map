# Changelog

## 1.0.0

- Prepared the first public release candidate.
- Added French, German, Spanish, Italian, Turkish, Portuguese, Hindi, Bengali, and Tamil.
- Expanded automatic language detection to all 12 supported interface languages.
- Added action-oriented command names: **Open storage map** and **Scan vault storage**.
- Excluded `storage-cache-v1.json` from scan totals automatically.
- Clarified that scanning is read-only while settings, cache, and explicitly exported reports may be written locally.
- Added an English-first publication README, Russian and Chinese quick guides, translation status, security policy, contributing guide, issue templates, CI, and automated release workflow.
- Added publication assets for BRAT and the official Community Plugins submission.

## 0.2.4

- The details panel no longer occupies space until an item is selected.
- Added a compact collapsible details panel and responsive table layout.
- Kept the Size column visible on narrow windows.

## 0.2.3

- Kept the folder-path tooltip inside the visible panel bounds.

## 0.2.2

- Replaced overlapping native tooltips with one managed popover.

## 0.2.1

- Softened warning colors and improved folder-path help and scan cancellation guidance.

## 0.2.0

- Added multilingual UI, themes, search, details, cache, scan comparisons, exports, responsive layout, and recommendations.

## 0.1.0

- Initial working release with vault scanning, treemap, tables, file types, and recommendations.

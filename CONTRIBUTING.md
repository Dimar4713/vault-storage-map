# Contributing

Bug reports, translations, documentation improvements, and pull requests are welcome.

## Development

```bash
npm install
npm run check
npm run build
```

## Pull requests

- Keep storage scanning local and read-only.
- Do not add telemetry or network access without an explicit, documented opt-in design review.
- Preserve all translation placeholders.
- Run the production build before submitting.
- Update the changelog for user-visible changes.

# Vault Storage Map

**Vault Storage Map** 是一个本地优先的桌面插件，用于直观显示库中哪些文件夹和文件占用了磁盘空间。

**开发：** AIMETON · Dimar4713。

## 功能

- 交互式矩形树图；
- 最大文件夹和文件排名；
- 搜索和大小筛选；
- 可折叠的紧凑详情面板；
- 系统、浅色和深色主题；
- 上次扫描缓存和扫描对比；
- 导出 Markdown、CSV 和 JSON；
- 针对大文件、`.obsidian`、Copilot 索引和附件的诊断建议；
- 12 种界面语言；
- 无遥测、无网络上传、无自动删除。

## 隐私

扫描为只读操作。插件只读取路径、大小、扩展名、修改日期和目录计数等文件系统元数据，不会为存储分析读取笔记正文，也不会通过网络发送库数据。

插件只保存设置、可选的本地扫描缓存，以及用户明确导出的报告。插件生成的缓存文件会自动从扫描总量中排除。

## 手动安装

从相应版本中下载 `main.js`、`manifest.json` 和 `styles.css`，复制到：

```text
<Vault>/.obsidian/plugins/vault-storage-map/
```

重新加载应用并启用插件。

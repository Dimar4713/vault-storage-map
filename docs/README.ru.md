# Vault Storage Map

**Vault Storage Map** — локальный плагин для настольной версии, который показывает, какие папки и файлы занимают место в хранилище.

**Разработка:** AIMETON · Dimar4713.

## Возможности

- интерактивная treemap-карта;
- рейтинги крупнейших папок и файлов;
- поиск и фильтры размера;
- компактная сворачиваемая панель подробностей;
- системная, светлая и тёмная темы;
- кэш последнего анализа и сравнение сканирований;
- экспорт Markdown, CSV и JSON;
- рекомендации по крупным файлам, `.obsidian`, индексам Copilot и вложениям;
- 12 языков интерфейса;
- отсутствие телеметрии, сетевой отправки данных и автоматического удаления.

## Конфиденциальность

Сканирование выполняется только для чтения. Анализируются пути, размеры, расширения, даты изменения и количество объектов. Содержимое заметок для анализа пространства не читается и не отправляется в сеть.

Плагин сохраняет настройки, необязательный локальный кэш и явно экспортированные пользователем отчёты. Собственный файл кэша исключается из результатов сканирования.

## Установка вручную

Скопируйте `main.js`, `manifest.json` и `styles.css` из релиза в:

```text
<Хранилище>/.obsidian/plugins/vault-storage-map/
```

Перезапустите приложение и включите плагин.

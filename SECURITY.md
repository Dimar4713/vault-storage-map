# Security policy

## Data handling

Vault Storage Map performs local filesystem metadata analysis. It does not include telemetry, analytics, accounts, remote APIs, or network requests.

Scanning is read-only. The plugin can write its settings, optional scan cache, and reports explicitly exported by the user.

## Reporting a vulnerability

Please report security-sensitive issues privately to the maintainer through the GitHub profile contact options rather than opening a public issue. Include reproduction steps, affected versions, and potential impact.
